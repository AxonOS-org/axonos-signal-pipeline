## Reproducing and changing vectors

These artifacts are generated by `tools/gen_test_vectors.py`, the single
source of truth, and verified by `tools/validate_vectors.py`. **Do not edit
the JSON or the generated `vectors.rs` by hand.** To change a vector:

1. edit `tools/gen_test_vectors.py`;
2. run `python3 tools/gen_test_vectors.py`;
3. commit **all** regenerated outputs — the JSON vector file, the synthetic
   fixture, the generated `crates/axonos-pipeline-core/tests/data/vectors.rs`,
   and the regenerated `SHA256SUMS` — in the **same commit**.

CI rejects any out-of-sync or unpinned change (atomic-update rule).
